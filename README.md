# 関数グラフ・シミュレーター

中学生の数学学習を支援するために開発された、インタラクティブなグラフ描画ツールです。
単一のHTMLファイルで動作するため、サーバー設定やビルドプロセスなしで利用可能です。

## 🎯 教育的狙い

1.  **「係数」の意味の視覚化**:
    *   $y = ax$ の $a$ が「傾き」であることを直感的に理解。
    *   $y = ax^2$ の $a$ の符号による形状変化を観察。
2.  **数式とグラフのリアルタイム連携**:
    *   操作に対する即座のフィードバック。
3.  **正しい数学表記**:
    *   小数は分数に変換（例: $0.5x \to \frac{1}{2}x$）。
    *   $y=0$ などの特殊なケースも正しく表示。

## 🛠 機能

*   比例、反比例、一次関数、二次関数に対応。
*   $y=0$ の適切な表示。
*   反比例の漸近線付近の描画改善。
*   1目盛りごとのグリッド線。

## 🚀 使い方

1.  `index.html` をブラウザで開くだけで動作します。
2.  GitHub Pages等にアップロードする場合も、このファイル1つで動作します。

## 🎨 技術スタック

*   HTML5 (Babel Standalone内蔵)
*   React 18
*   Recharts
